# AI Career Tool — Skill Analyzer

A Python tool that analyzes your skills and suggests
the best career paths for bioinformatics and tech professionals.

## What it does
- Collects your skill ratings across 10 key areas
- Matches your profile to 8 career paths
- Identifies skill gaps for each career
- Generates personalized action plan
- Visualizes career match scores
- Exports detailed career report

## Career Paths Analyzed
- Bioinformatician
- Data Scientist
- ML Engineer
- Computational Biologist
- Clinical Data Analyst
- Biotech Data Analyst
- Research Scientist
- Full Stack Developer
